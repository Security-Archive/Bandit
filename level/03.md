# Bandit Level 2 → Level 3
## Level Goal
The password for the next level is stored in a file called `--spaces in this filename--` located in the home directory

다음 레벨을 위한 비밀번호는 홈 디렉토리에 위치한 `--spaces in this filename--`라는 파일에 저장되어 있습니다.

## Solution
`ls`로 확인해보면 파일명에 스페이스가 포함되어있다. 그리고 이걸 읽으려고 하면 역시나 제대로 인식하지 못한다.

### 공백이 포함된 파일명을 그대로 인식하지 못하는 이유
리눅스 셸은 **공백(space)**을 **인자(argument) 구분자**로 사용한다. <br />
따라서 공백이 포함된 파일명을 하나의 파일이 아니라 여러 개의 인자로 분리되어 해석된다.

### 해결 방법
1. 백슬래시(`\`)로 공백 이스케이프
   ```sh
   cat ./--spaces\ in\ this\ filename--
   ```
   - `\`는 다음 문자를 문자 그대로 해석하도록 만든다.
   - 공백을 구분자가 아닌 파일명의 일부로 처리한다.

2. 따옴표로 전체를 감싸기
   ```sh
   cat "./--spaces in this filename--"
   ```
   - 따옴표 안의 공백은 분리되지 않음
   - 가장 가독성이 좋은 방법
